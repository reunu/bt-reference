# Redis Reference

Documenting Redis DB

## Redis instance

You can connect to the [redis](https://redis.io) instance via:

| **IP**      | **Port** |
|-------------|----------|
| 192.168.7.1 | 6379     |

You can use the following command locally:

```
redis-cli -h 192.168.7.1 -p 6379
```
